How to Install ChemVantage

ChemVantage is certified to comply with 1EdTech specifications for LTI Advantage Complete (version 1.3). Due to potential security vulnerabilities, we no longer support LTI version 1.1 registrations. Unfortunately, the registration process is highly dependent on which LMS you are using. To get started, please follow the instruction below.

Does your LMS support LTI Dynamic Registration?

  • IF YES: the LMS administrator should go to the LTI registration page in your LMS and use the Dynamic Registration URL: https://www.chemvantage.org/lti/registration
    They will submit a brief form, activate the ChemVantage app, and configure which organizational units in your institution can access it. That's all!

  • IF NO: please select your LMS from the following options for detailed installation instructions

Installation instructions for Blackboard Learn

The institution's account administrator should perform the following steps:
  1. Browse to https://www.chemvantage.org/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. Go to System Admin | Integrations: LTI Tool Providers | Register LTI 1.3 Tool
  3. Enter the Client ID: be1004de-6f8e-45b9-aae4-2c1370c24e1e
  4. Make a copy of the deployment_id and set Tool status: Approved
  5. Institution Policies: Send Role, Name, Email; Allow Grade Service and Membership Service
  6. Submit
  7. Use the tokenized link from the email in step 1 to register your deployment_id with ChemVantage. If the link expired, you can simply repeat step 1 to get another one.
  8. Go back to the LTI Tool Providers page, and from the dropdown menu on the ChemVantage app select Manage Placements
  9. Click Create Placement
    • Label: ChemVantage
    • Description: ChemVantage is an Open Educational Resource for General Chemistry
    • Handle: (any unique string)
    • Availability: Yes
    • Deep Linking Tool (do not allow student access)
    • Target Link URI: https://www.chemvantage.org/lti/deeplinks
    • Custom Parameters (leave blank)
    • Submit
The course instructor can then create any ChemVantage assignment:
  1. Go to the course | Content | Build Content | ChemVantage
  2. Select the ChemVantage assignment type and topic

Installation instructions for D2L Brightspace

First of all, Brightspace supports LTI Advantage Dynamic Registration! The Brightspace documentation can be found here. We strongly encourage you to try that first (see above), but the institution's account administrator still has the option of manual registration:
  1. Browse to https://www.chemvantage.org/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. In Brightspace, ensure the Manage LTI Advantage Tool Registrations and Manage LTI Advantage Tool Deployments permissions are set at the organization level.
  3. From the Admin Tools menu, click Manage Extensibility. In the LTI Advantage tab, click Register Tool.
  4. Choose Standard registration, click Register.
  5. Tool Name: ChemVantage
  6. Description: ChemVantage is an Open Educational Resource for college-level General Chemistry
  7. Domain: https://www.chemvantage.org Note: Domains can only be used in one enabled registration at a time. If you attempt to register a tool with a domain that is already in use, you will be prompted to disable the existing registration.
  8. Redirect URIs: https://www.chemvantage.org/lti/launch https://www.chemvantage.org/lti/deeplinks
  9. OpenID Connect Login URL: https://www.chemvantage.org/auth/token
  10. Target Link URI: https://www.chemvantage.org/lti/launch
  11. LTI version: LTI 1.3
  12. Keyset URL: https://www.chemvantage.org/jwks
  13. Tool URL: https://www.chemvantage.org/lti/launch
  14. Extensions: Deep Linking, Assignment and Grade Services, Names and Role Provisioning Services
  15. Deep Linking Content Selection URL: https://www.chemvantage.org/lti/deeplinks
  16. Under Roles, choose to Context role information. Sending Institutional role information is optional. Setting the Deployment security to Anonymous will work; however, instructors will be unable to verify or synchronize ChemVantage scores to the Brightspace grade book using this setting.
  17. Substitution Parameters: resource_link_id_history=$ResourceLink.id.hisory
  18. Custom parameters: (none)
  19. Click Register
  20. Review the configuration details to obtain the client_id and deployment_id, and click the tokenized link in the email confirmation from step 1 to register them with ChemVantage. If the link expired, you can simply repeat step 1 to get another one.

Installation instructions for Instructure Canvas

The institution's account administrator should perform the following steps:
  1. Browse to https://www.chemvantage.org/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. Configure a new LTI Developer Key for your Canvas Account (see detailed instructions here)
  3. Use the following Key Settings:
    • Key Name: ChemVantage
    • Owner Email: admin@chemvantage.org
    • Redirect URIs:
      https://www.chemvantage.org/lti/launch
      https://www.chemvantage.org/lti/deeplinks
    • Configure Method: Enter URL
    • JSON URL: https://www.chemvantage.org/lti/registration?UserRequest=config
  4. Click Save
  5. Copy or write down the client_id and deployment_id created in step 2. This is the tricky part, because Canvas doesn't make it easy. Canvas uses the developer key as the client_id, so it can be viewed from the list of developer keys. It is a numeric value that looks something like 32570000000000041. The deployment_id can be found in Settings | Apps | App Configurations by opening the settings menu for ChemVantage. It is a compound value that consists of a number and a hex string separated by a colon and looks something like 10408:7db438070728c02373713c12c73869b3af470b68.
  6. Add ChemVantage as an External App to your account using the client_id created in step 2 (see detailed instructions here)
  7. Click the tokenized link in the email confirmation from step 1 to register the client_id and deployment_id with ChemVantage. If the link expired, you can simply repeat step 1 to get another one.
The course instructor can then create any ChemVantage assignment as follows:
  1. Create a new Canvas assignment with the following recommended parameters:
  2. Name: (as appropriate, e.g. Quiz - Heat and Enthalpy)
  3. Points: 10 for quiz or homework; 5 for video, poll or reading assignment; 100 for placement exam or practice exam
  4. Submission Type: External Tool
  5. External Tool URL: Find ChemVantage or enter https://www.chemvantage.org/lti/launch
  6. Save or Save and Publish
  7. When you launch the assignment, you may use the links on the Instructor page to customize it for your class.

Installation instructions for Moodle

First of all, Moodle supports LTI Advantage Dynamic Registration! The Moodle documentation can be found here. We strongly encourage you to try that first (see above), but the institution's account administrator still has the option of manual registration:
  1. Browse to https://www.chemvantage.org/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  2. As an admin user, go to "Site administration > Plugins > External tool > Manage tools"
  3. Click the link to "Configure a tool manually"
  4. External Tool Configuration
    • Tool Name: ChemVantage
    • Tool URL: https://www.chemvantage.org/lti/launch
    • Tool Description: ChemVantage is an Open Educational Resource for teaching and learning college-level General Chemistry
    • LTI version: LTI 1.3
    • Public Key Type: Keyset URL
    • Public Keyset: https://www.chemvantage.org/jwks
    • Initiate Login URL: https://www.chemvantage.org/auth/token
    • Redirection URIs:
      https://www.chemvantage.org/lti/launch
      https://www.chemvantage.org/lti/deeplinks
    • Custom parameters: (none)
    • Check 'Supports Deep Linking'
    • Content Selection URL: https://www.chemvantage.org/lti/deeplinks
    • Services | LTI Assignment and Grade Services: select Use for grade sync only
    • Services | LTI Names and Role Provisioning: select Use this service
    • Privacy | check Force SSL
    • Save Changes
  5. When you have finished the configuration, Moodle generates a preconfigured tool. You must activate it and then click 'View configuration details'. When you have these details in hand, including the client_id and deployment_id, click the tokenized link in the email confirmation from step 1 to register the configuration details with ChemVantage. If the link expired, you can simply repeat step 1 to get another one.
The course instructor can then create any ChemVantage assignment as follows:
  1. In any course, turn Edit Mode on and click 'Add an activity or resource'
  2. Select External tool
  3. Activity Name: (as appropriate, e.g. Quiz - Heat and Enthalpy)
  4. Preconfigured Tool: select ChemVantage
  5. Click 'Select Content' and choose the assignment type and topics from the ChemVantage popup window
  6. Configure Grade and Privacy settings as desired
  7. When you launch the assignment, you may use the links on the Instructor page to customize it for your class.

Installation instructions for PowerSchool Schoology

  1. Click the App Center icon, then select App Center.
  2. Locate ChemVantage and click the title
  3. Click Install LTI 1.3 App > I Agree > Continue > Add to Organization
  4. You will be directed to Organization Apps to continue the installation.
  5. Click the Install/Remove button for ChemVantage
  6. Select All Users, All Courses, or areas where you want to install ChemVantage. Then Submit.
  7. Click Configure and copy the Deployment ID for ChemVantage. It should be two 10-digit numbers separated by a hyphen. You will need it to complete the registration.
  8. Browse to https://www.chemvantage.org/Registration.jsp and complete a brief form to provide your contact information. We will send an email confirmation with a tokenized registration link.
  9. Click the tokenized link in the email confirmation to register the Deployment ID and other configuration details with ChemVantage. If the link expires, you can simply repeat the previous step to get another one.
The course instructor can then create any ChemVantage assignment as follows:
  1. In any course, click 'Add Materials' and select ChemVantage
  2. Choose the assignment type and topics from the ChemVantage popup window
  3. Use the Schoology assignment settings to edit the Grading Configuration and points
  4. When you launch the assignment, you may use the links on the ChemVantage Instructor page to customize it for your class.
